Short Code / Long Code / Virtual Number

Services

  • Short Code is a five digit number that start with 5. It is used to receive incoming messages. It is valid only in the geographical limits of a country for receiving incoming messages. The cost of pushing messages to such numbers is comparably higher than that of normal plans.
  • Long Code is a 10 digit number that follows the international numbering plan and is also used to receive incoming messages. The cost of sending messages to this number is a normal STD SMS charge. A long code is also known as a Virtual Mobile Number.

Benefits

  1. The Short Code and Long codes made two-way communication possible with the masses. All campaigns can be made and the response can be captured and measured if the incoming part in integrated. This leads to monetary optimizations with respect to the funds required for the campaign.
  2. It also completes the communication loop for the customer and the customer is left feeling satisfied and complete.
